# 🏋️ Iron House – Full Stack Gym Management System

Iron House is a full-stack gym management web application built using the MERN stack. It provides a complete digital platform for gym members and administrators, featuring secure authentication, membership management, AI-powered assistance, and real-time push notifications.

---

## 🚀 Features

### 👤 Authentication
- JWT Authentication
- Google OAuth Login
- Forgot & Reset Password via Email

### 💳 Membership Management
- Browse available membership plans
- Purchase and manage memberships
- Membership status tracking

### 📅 Trial Booking
- Book free gym trial sessions
- Admin management of trial requests

### 🤖 AI Fitness Assistant
- AI-powered fitness chatbot
- Workout and fitness guidance

### 🔔 Push Notifications
- Firebase Cloud Messaging (FCM)
- Browser push notifications
- Background and foreground notification support

### 👨‍💼 Admin Panel
- Manage memberships
- Manage bookings
- User management

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- JWT Authentication
- Passport Google OAuth

### Database
- MongoDB Atlas
- Mongoose

### Cloud & Services
- Firebase Cloud Messaging
- Render (Backend Deployment)
- Vercel (Frontend Deployment)

## 📌 Key Highlights

- Full-stack MERN architecture
- Secure authentication with JWT & Google OAuth
